# 定義 03：初始值 — 單點離散化 `[0 : 1]`

## 1. 陳述式

除 Chebyshev 五點離散化外，本專案允許 **單點（一次取樣）** 的退化離散分佈：

```
T = [0 : 1]
```

即：

- 僅 **1 個樣本值**：`v₁ = 0`
- 對應機率：`p₁ = 1`
- **期望值**：`E[T] = 1 × 0 = 0`

## 2. 語意：初始值

此表示法通常用於 **初始值（initial value）** 情境，例如：

- 專案網路的 **起始節點**（node 0）活動時間為 0
- 靜態環境下 `T = 0`，隨機環境下不再展開為五點常態近似，而直接視為 **確定為 0 的退化隨機變數**

**理由：** 期望值為 0 時無需（也不應）以 Chebyshev 五點在 0 附近展開隨機性；單點 `[0 : 1]` 語意清楚，且計算最簡。

## 3. 陳述式：與五點離散化的關係

| 條件 | 離散化結果 | 方法標記 |
|------|------------|----------|
| `Value > 0` | `[v₁…v₅ : p₁…p₅]`（Chebyshev 五點） | `chebyshev_5` |
| `Value = 0` | `[0 : 1]`（單點初始值） | `initial` |

## 4. 陳述式：矩陣表示

```
T = [ 0 ]    （第一列：樣本值）
    [ 1 ]    （第二列：機率）
```

記法：`T = [0 : 1]`

## 5. Python 契約

```python
discretization(0.0)
# values         -> (0.0,)
# probabilities  -> (1.0,)
# notation()     -> [0 : 1]
# expected_value -> 0.0
# method         -> "initial"
```

**實作：** `api/analysis/discretization.py` 在 `Value == 0` 時直接回傳單點結果，不呼叫 Chebyshev 五點。
