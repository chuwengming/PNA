# 定義 02：Chebyshev 五點離散化技術

## 1. 方法概述

**Chebyshev 離散化**（Chebychev discretization）將連續常態隨機變數 `X ~ N(μ, σ²)` 近似為 **5 點離散分佈**，使：

1. **期望值完全匹配**：`E[X] = μ`（即輸入的 `Value`）
2. 支撐點對稱分布於均值兩側，間距為 `σ` 的整數倍
3. 計算簡單、適合專案網路的大規模反覆運算

本專案採此作為 **預設且唯一** 的離散化方法（`discretization()` 內部實作）。

> 參考：專案文件 `docs/source/` 中關於隨機環境取樣與離散化之說明；  
> 文獻上相近方法見 Keefer & Bodily (1983) 離散近似、Miller & Rice (1983) Gauss–Hermite 五點法；  
> Chebyshev 五點法以 Chebyshev 不等式為動機，在 `μ ± 2σ` 內至少涵蓋 75% 機率質量。

## 2. 支撐點（Support Points）

取標準偏移量 `k ∈ {−2, −1, 0, 1, 2}`，定義 5 個樣本值：

```
vᵢ = μ + kᵢ · σ
```

| 索引 i | kᵢ | 樣本值 vᵢ |
|--------|-----|-----------|
| 1 | −2 | μ − 2σ |
| 2 | −1 | μ − σ |
| 3 | 0 | μ |
| 4 | +1 | μ + σ |
| 5 | +2 | μ + 2σ |

## 3. 機率權重（Probabilities）

Chebyshev 五點離散化採用以下 **固定機率向量**（與 `μ`、`σ` 無關）：

```
P = [p₁, p₂, p₃, p₄, p₅] = [0.05, 0.25, 0.40, 0.25, 0.05]
```

## 4. 陳述式：期望值守恆（可證）

```
E[X] = Σᵢ pᵢ · vᵢ
     = Σᵢ pᵢ · (μ + kᵢσ)
     = μ · Σᵢ pᵢ  +  σ · Σᵢ pᵢ kᵢ
```

代入 `Σ pᵢ = 1` 及 `Σ pᵢ kᵢ = 0.05(−2) + 0.25(−1) + 0.40(0) + 0.25(1) + 0.05(2) = 0`：

```
E[X] = μ · 1 + σ · 0 = μ  ✓
```

因此只要 `vᵢ = μ + kᵢσ`，**不論 σ 取何正值，期望值恆等於 `Value`（μ）**。

## 5. 陳述式：二維矩陣表示

離散化結果以 **2 × 5 矩陣** 表示（**第一列樣本值、第二列機率**，與 `docs/source/stochastic network analyisis.pdf` 一致）：

```
       ┌                                                                                ┐
T  =   │  v₁      v₂      v₃      v₄      v₅   │  =  │ μ−2σ   μ−σ    μ     μ+σ   μ+2σ  │
       │  p₁      p₂      p₃      p₄      p₅   │     │ 0.05   0.25   0.40   0.25   0.05 │
       └                                                                                ┘
```

**記法：** `T = [v₁, v₂, v₃, v₄, v₅ : p₁, p₂, p₃, p₄, p₅]`

## 6. 數值範例

**輸入：** 靜態時間 `T = 2`（即 `μ = 2`），預設 `σ = μ/6 = 1/3 ≈ 0.3333`。

| i | pᵢ | kᵢ | vᵢ |
|---|-----|-----|-----|
| 1 | 0.05 | −2 | 1.3333 |
| 2 | 0.25 | −1 | 1.6667 |
| 3 | 0.40 | 0 | 2.0000 |
| 4 | 0.25 | +1 | 2.3333 |
| 5 | 0.05 | +2 | 2.6667 |

**驗證期望值：**

```
E[T] = 0.05×1.3333 + 0.25×1.6667 + 0.40×2.0 + 0.25×2.3333 + 0.05×2.6667
     = 2.0000  ≈  Value  ✓
```

**完整記法：**

```
T = [1.3333, 1.6667, 2.0000, 2.3333, 2.6667 : 0.05, 0.25, 0.40, 0.25, 0.05]
```

## 7. 非負性約束

活動時間不可為負，故需 `v₁ = μ − 2σ ≥ 0`，即 **`σ ≤ μ/2`**。

- 預設 `σ = μ/6` 時恆滿足 `μ ≥ 2σ`，五點皆為正。
- 若使用者傳入較大的 `std_dev`，實作會自動將 `σ` 上限設為 `μ/2`（見 `_cap_sigma_for_non_negative_support`），以維持非負支撐點與 `E[T] = μ`。

## 8. Python 介面

```python
from api.analysis.discretization import discretization

result = discretization(2.0)
# result.values         -> [1.333..., 1.667..., 2.0, 2.333..., 2.667...]
# result.probabilities  -> [0.05, 0.25, 0.40, 0.25, 0.05]
# result.mean           -> 2.0
# result.as_matrix()    -> [[v1, ..., v5], [0.05, 0.25, 0.40, 0.25, 0.05]]
# result.notation()     -> [1.33, 1.67, 2, 2.33, 2.67 : 0.05, 0.25, 0.4, 0.25, 0.05]
# result.expected_value -> 2.0
```

可選參數 `std_dev` 覆寫預設 `σ = Value / 6`。

## 9. 與其他離散化方法之比較（備註）

| 方法 | 特點 | 本專案 |
|------|------|--------|
| **Chebyshev 五點** | 固定機率、對稱支撐點，`E[X]=μ` 恆成立 | ✅ 採用 |
| Miller–Rice (Gauss–Hermite) | 匹配更多矩，節點非等距 | 未採用 |
| Keefer–Bodily 三點 | 僅 3 點，精度較低 | 未採用 |

若未來需切換方法，應新增獨立定義文件與函式，不破壞本 Chebyshev 契約。
